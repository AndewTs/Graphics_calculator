# Graphics_calculator
My first project in 2025 year in Yandex Liceum.


# Основные функциональные требования  
1) #### Графические построения

 - Умеет строить на одной координатной плоскости несколько функций,
введённых с клавиатуры  
- Графически решает неравенства с закрашиванием
областей, в которых находятся решения

2) #### Внешний вид

 - Выбор цвета и стиля линии (сплошная, пунктирная, через точку с запятой)
 -  Настройка прозрачности линии

3) #### Работа с файлами

 - Через диалоговое окно можно выбрать .txt/.csv файл с набором функций;
все функции из файла строятся на одной плоскости  
- Поддерживается экспорт всех введённых функций в .txt/.csv  
-  Сохранение изображения графика (в формате .png, .jpeg)

4) #### Анализ функции

 - Краткий анализ: точки пересечения с осями x и y, проверка
чётности/нечётности  
- Производная функции (символьно или численно)  
- Определённый интеграл на заданном промежутке  
-  Нахождение экстремумов на указанном отрезке

5) #### Работа с БД

 - В отдельной БД хранятся несколько последних запросов пользователя  
 - Просмотр истории запросов и восстановление выбранного запроса
(подстановка выражения в поле ввода и/или повторное построение)


# Архитектура
#### Модуль построения графиков

 - Построение графиков функций, управление слоями графиков и настройками
внешнего вида

#### Модуль анализа функции

 - Парсер, символьные и численные операции: производная, интеграл,
корни, экстремумы, тесты на чётность

#### Модуль работы с файлами

 - Диалоги открытия/сохранения, экспорт функций

#### Модуль работы с БД

 - Интерфейс для записи и чтения истории запросов, восстановление
записей


# Дизайн
#### Главное окно

 - Поле для ввода функции и меню

#### Координатная плоскость

 - Виджет, отвечающий за вычисления и визуализацию графиков и
закрашенных областей


# Горячие клавиши

 - **Enter** --- построение введённой функции  
 - **Ctrl + O** --- загрузить функции из файла  
 - **Ctrl + S** --- сохранить введённые функции в файл  
 - **Ctrl + A** --- анализ функции  
 - **Ctrl + H** --- посмотреть историю запросов
